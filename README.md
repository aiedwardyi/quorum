# Quorum

Multi-AI group chat for consensus. Ask a question, AI models discuss it with each other, you steer the conversation, they converge on an answer.

## Stack

- Next.js 14+ / TypeScript / Tailwind CSS / shadcn/ui
- Google Vertex AI (Gemini) + Perplexity API
- SSE streaming

## Setup

npm install
gcloud auth application-default login
# Add your Perplexity API key to .env
npm run dev
